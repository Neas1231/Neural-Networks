Комментарий к структуре файлов:
api - директория содержащая созданную API, описанную в отчёте Report.ipynb и документации
data - данные разделённые и вынесенные для обучения модели yolov5
Result_A - данные полученные из предобработок в модуле А
static - изображения, используемые в отчете Report.ipynb и в остальных файлах
unit - директория с файлами .py расширения в которых проводились тесты функций для API
yolov5 - модель yolov5 веса которой я обучал и модель которая исполняет задачу детекции автомобильных номеров
last.pt - веса модели yolov5 которые были обучены выделять наши номера из фотографий
model.pickle - интерпретация модели yolov5 в формате pickle
Readme.txt - комментарии к модулю
Report.ipynb - отчёт по всей работе в Модуле Б
Документация к АПИ.docx - документация к созданному API и функциям в этом API
